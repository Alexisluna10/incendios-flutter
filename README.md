# Seguridad infantil - Sistema de prevención para menores de edad. 

Este repositorio contiene el código fuente de **Seguridad infantil**, una aplicación desarrollada como una solución de pruebas para evitar accidentes en entornos que un menor de edad no pueda controlar o no tiene conocimiento. 

El sistema fue diseñado y construido de manera colaborativa por un equipo de desarrollo de 3 personas, enfocándose en la experiencia del usuario, entendimiento y accesibilidad.
