# Child Safety - Prevention System for Minors

This repository contains the source code of **Child Safety**, an application developed as a test solution to prevent accidents in environments that a minor cannot control or is not aware of.

The system was designed and built collaboratively by a development team of 3 people, focusing on user experience, understanding, and accessibility.
